
                        ******************************
			Дополнительные значки uTorrent
                        ******************************

    ВНИМАНИЕ!!! Если у Вас в системе отключён показ расширений зарегистрированных типов файлов, 
    то написанное ниже, следует применять без учёта расширения .ico (т.е. рассматривать НЕ файлы 
    main.ico и tray.ico, а файлы main и tray).

 1. Переименовать любой из файлов main_tray_XX.ico в:
    main.ico - для изменения значка в главном окне программы
    tray.ico - для изменения значка программы в трей
    Полученные файлы скопировать в папку с программой (это папка на уровень выше текущей),
    заменить, если уже существуют. 
    *Удаление файлов main.ico и/или tray.ico в папке с программой, вернёт стандартные значки

 2. Переименовать любой из файлов maindoc_XX.ico в maindoc.ico и заменить полученным файлом
    одноимённый в папке с программой для изменения значка ассоциированных с uTorrent типов файлов.



                           *************************
			   Additional uTorrent icons 
                           *************************

    ATTENTION!!! If you have disabled display of extensions of registered file types on your system, 
    the following should be applied without taking into account the .ico extension (i.e. to consider NOT files 
    main.ico and tray.ico but the main and tray files).

 1. Rename any of the main_tray_XX.ico files to:
    main.ico - to change the icon in the main program window
    tray.ico - to change the program icon in the tray
    Copy the resulting files to the folder with the program (this folder is one level higher than the current one),
    replace if files already exist. 
    *Deleting the main.ico and/or tray.ico files in the program folder will return the standard icons

 2. Rename any of the files maindoc_XX.ico to maindoc.ico and replace with the resulting file
    the same name in the folder with the program to change the icon of the file types associated with uTorrent.
